# Iteration 3: The "Paper & Ink" Release

**Version:** 0.1.0
**Date:** 2025-11-25
**Theme:** Design System & Navigation

## Summary
Iteration 3 focused on establishing the visual identity and structural foundation of the ASDLC knowledge base. We implemented a custom "Industrial/Spec-Sheet" design system ("Paper & Ink"), overrode default Astro styles, and finalized the global navigation wrapper.

## Completed Work

### Features (PBIs)
*   **PBI-0005: Navigation Wrapper** - Implemented global Header/Footer, System Status Banner, and layout structure.
*   **PBI-0006: Design System Override** - Established typography (Archivo + B612 Mono) and basic layout density.
*   **PBI-0007: Design System Colors** - Implemented the "Paper & Ink" color palette, "Safety Orange" branding, and functional status colors.
*   **PBI-0008: Navigation Regressions** - Fixed regressions caused by design system updates, ensuring navigation and banners align with the new aesthetic.

### Defects Fixed
*   **DEFECT-001: Banner Placement** - Moved Alpha banner to content block.
*   **DEFECT-002: Banner Style** - Fixed banner borders and branding.
*   **DEFECT-003: Banner Colors** - Corrected banner colors to use Warning palette.
*   **DEFECT-004: Heading Spacing** - Fixed letter spacing for uppercase headings.
*   **DEFECT-005: Multiple H1s** - Fixed heading hierarchy on Guardrails page.
